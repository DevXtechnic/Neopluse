## Pair Extraordinaire Collaboration

I’m pairing with `kmkrishnacodes` on this repository to earn GitHub’s **Pair Extraordinaire** badge. The idea is simple:

1. Work together on a feature while each person contributes code or a review.
2. Make sure every co-authored commit contains the trailer  
   `Co-authored-by: kmkrishnacodes <kmkrishnaacer@gmail.com>`
3. Merge the pull request that carries the co-authored commit into the default branch.

This README update documents the collaboration so reviewers know the badge is the target and where to look for the co-authorship trailer.
